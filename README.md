# MATRIX Logistics

Лендинг компанії з імпорту вантажів з Китаю в Україну.

## Що це

Односторінковий сайт з інтерактивним 3D-глобусом Землі, на якому в реальному часі рухаються літаки й кораблі за справжніми торговими маршрутами Шеньчжень → Одеса (через Суецький канал і в обхід Африки через мис Доброї Надії).

## Особливості

- **3D-глобус на Three.js** — нічна Земля з вогнями міст, обертається мишею, з атмосферою
- **Анімовані маршрути** — літаки між Києвом, Пекіном і Шеньчженем; кораблі двома морськими шляхами
- **Картки тарифів** — авіа, морська, залізнична доставка з реальними фото
- **Контактна модалка** — WhatsApp, Telegram, форма заявки
- **Темна / світла теми** з ефектом «matrix rain» у темному режимі
- **Калькулятор вартості** доставки
- Адаптивна верстка під десктоп і мобільні

## Структура

```
index.html         — основна розмітка
style.css          — стилі (теми, секції, картки, модалка)
script.js          — інтерактив (модалка, тема, форма, matrix-ефект)
globe.js           — 3D-глобус, маршрути, кораблі/літаки
images/            — фото для карток і ассети
```

## Технології

Чистий HTML/CSS/JavaScript без фреймворків. 3D-частина — на [Three.js r128](https://threejs.org/), підключається з CDN. Жодних залежностей, жодного збирання — все працює напряму в браузері.

## Як запустити локально

Просто відкрийте `index.html` у будь-якому сучасному браузері.

> Деякі речі (текстура нічної Землі з CDN) краще працюють через локальний сервер. Якщо є Python — у папці сайту запустіть `python3 -m http.server`, далі відкрийте `http://localhost:8000`.

## Деплой

Сайт повністю статичний — підходить будь-який хостинг статики:

- **GitHub Pages** — Settings → Pages → Branch `main` → `/ (root)` → Save
- **Netlify** — імпортувати репозиторій, deploy
- **Vercel** — те саме
