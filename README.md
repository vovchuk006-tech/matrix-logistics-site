# MATRIX Landing

Готовий статичний сайт для GitHub.

## Що всередині
- `index.html`
- `style.css`
- `script.js`

## Як запустити локально
Просто відкрий `index.html` у браузері.

## Як завантажити на GitHub
1. Створи новий репозиторій на GitHub
2. Завантаж туди ці 3 файли
3. За потреби увімкни GitHub Pages:
   - Settings
   - Pages
   - Build and deployment
   - Deploy from a branch
   - Branch: `main` / root

## Що ще можна доробити
- підключити реальні фото замість декоративних градієнтів
- зробити випадаючі меню в хедері
- підключити форму до Telegram / email / CRM
- додати favicon, OG tags, анімації
